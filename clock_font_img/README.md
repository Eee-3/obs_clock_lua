# 时钟字体图片生成器

用于生成 OBS 日期及毫秒级时钟脚本所需的数字和分隔符图片。

## 功能特性

- **交互式字体选择**：支持实时搜索系统字体
- **自动优化字体大小**：自动计算最佳字体尺寸以适应目标图片
- **自定义输出**：可设置输出目录、图片尺寸、生成字符
- **跨平台支持**：支持 Windows、macOS、Linux

## 前置依赖

### 系统要求

- **Python**: >= 3.12

### Python 依赖

| 依赖包           | 版本要求  | 说明               |
| ---------------- | --------- | ------------------ |
| `pillow`         | >= 10.0.0 | 图片处理库         |
| `prompt-toolkit` | >= 3.0.0  | 交互式命令行界面库 |

## 安装步骤

### 方式一：使用 uv（推荐）

本项目使用 [uv](https://docs.astral.sh/uv/) 作为包管理器。

```bash
# 进入项目目录
cd clock_font_img

# 安装依赖（uv 会自动创建虚拟环境并安装依赖）
uv sync
```

### 方式二：使用 pip

```bash
# 进入项目目录
cd clock_font_img

# 创建虚拟环境
python -m venv .venv

# 激活虚拟环境
# Windows (PowerShell):
.\.venv\Scripts\Activate.ps1
# Windows (CMD):
.\.venv\Scripts\activate.bat
# macOS/Linux:
source .venv/bin/activate

# 安装依赖
pip install pillow prompt-toolkit
```

## 使用方法

### 运行程序

```bash
# 使用 uv
uv run main.py

# 或激活虚拟环境后直接运行
python main.py
```

### 交互式操作流程

1. **选择字体**
   - 程序会自动扫描系统字体
   - 输入关键字搜索字体
   - 使用 `↑` `↓` 键选择字体
   - 按 `Enter` 确认选择

2. **设置输出目录**
   - 默认值：`../obs_clock`
   - 图片将输出到脚本目录下的 `obs_clock` 文件夹

3. **设置图片尺寸**
   - 宽度默认：`22` 像素
   - 高度默认：`30` 像素

4. **设置生成字符**
   - 默认值：`0123456789:/.`
   - 包含数字 0-9、冒号、斜杠、小数点

5. **确认生成**
   - 输入 `y` 确认生成
   - 输入 `n` 取消操作

### 快捷键

| 按键      | 功能                 |
| --------- | -------------------- |
| `↑` / `↓` | 在字体列表中移动选择 |
| `Enter`   | 确认当前选择         |
| `ESC`     | 取消并退出           |
| 直接输入  | 搜索过滤字体         |

## 输出文件

生成的图片文件命名规则：

| 字符      | 文件名            | 用途                   |
| --------- | ----------------- | ---------------------- |
| `0` - `9` | `0.png` - `9.png` | 数字显示               |
| `:`       | `and.png`         | 时间分隔符（时:分:秒） |
| `/`       | `slash.png`       | 日期分隔符（年/月/日） |
| `.`       | `dot.png`         | 毫秒分隔符（秒.毫秒）  |

## 注意事项
1. **图片尺寸**：默认尺寸 22×30 像素与 OBS 脚本匹配，修改后需同步调整脚本中的布局参数
2. **输出目录**：确保有写入权限，目录不存在时会自动创建
